How it Works

The bookmark acts as a button that runs the above javascript when you press it. The code itself fetches all the URLs the page has requested recently, filters them for a specific request we want, the request to the Google Maps API. We then make the request ourselves which exposes useful information such as the longitude and latitude which we can then use to make a Google search.
